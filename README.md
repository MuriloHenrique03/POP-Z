#Aprensentação

A página POP-Z é luagar onde você porde encontrar astistas do mundo pop, como: Beyoncé, Rhianna, Taylor Swift etc. 
Apagina conta com o nome do(a) artista em destaque e quando clicado é redirecionado para o perfil do **spotfy**, logo em seguida vem o nao de nascimento da(o) artista
e depois vem as músicas mais famosas que assim que você clicar será redirencionado diretamente para o **spotify**, a mesma coisa acontece com os os albuns mais famosos.
